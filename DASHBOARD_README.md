# HERDMind Session Plan Dashboard

## Overview

An interactive dashboard to visualize all 100 possible session plan combinations, showing which 27 combinations have plans and which 73 are missing.

## Files

- `dashboard.html` - The main dashboard application (single HTML file with embedded JavaScript)
- `dashboard_data.json` - Data file containing all 100 combinations (generated by Python script)
- `generate_dashboard_data.py` - Python script that generates the JSON data from the CSV

## Setup

1. **Generate the data file** (if not already done):
   ```bash
   python generate_dashboard_data.py
   ```
   This creates `dashboard_data.json` from `HERDMind_Session_Plans.csv`

2. **Open the dashboard**:
   - Simply open `dashboard.html` in a web browser
   - Or serve it via a local web server (recommended):
     ```bash
     # Python 3
     python -m http.server 8000
     
     # Then open: http://localhost:8000/dashboard.html
     ```
